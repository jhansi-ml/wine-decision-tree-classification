# 🍷 Wine Classification using Decision Tree

## 📌 Project Overview
This project uses a Decision Tree classifier to predict wine classes based on chemical features.

## 📊 Dataset
- Source: sklearn wine dataset
- Samples: 178
- Features: 13

## ⚙️ Model
- Algorithm: Decision Tree
- Criterion: Entropy
- Max Depth: 3
- Min Samples Leaf: 7

## 📈 Results
- Train Accuracy: 0.937
- Test Accuracy: 0.917
- Cross Validation Accuracy: 0.910
